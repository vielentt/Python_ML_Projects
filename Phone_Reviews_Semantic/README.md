# Семантический анализ отзывов телефонов

*Для проекта использовался датасет на русском* [Phone Reviews](https://www.kaggle.com/datasets/theovall/phonereviews)


Была проведена обработка данных: удалены пропущенные значения, дубликаты, выявлены и удалены аномалии в данных оценки `(> 5 и < 1)` и данные с большими `(> 2100 символов)` отзывами.

С помощью `plotly.express` была произведена визуализация разброса оценок и выведена гистограмма распределения сантимента отзывов (позитивные с оценкой > 3, негативные < 3). С помощью `worldcloud` визуализированы облака слов: общее на датасет и для позитивных и негативных отзывов.

Данные были очищены от пунктуации. Далее была произведена очистка от стоп-слов, данные приведены к нижнему регистру и создан мешок слов с помощью `CountVectorizer`.

Для классификации была использована логистическая регрессия `LogisticRegression` библиотеки `sklearn` и выведены метрики с помощью `classification_report`.

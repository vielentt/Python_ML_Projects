# Предсказание стоимости недвижимости в Москве

Датасет состоит из реальных, актуальных на 2023 год данных. Необходимо было провести разведочный анализ данных и построить модель, предсказывающую стоимость квартир по этим данным.

Для проекта использовались библиотеки `pandas, numpy, matplotlib, seaborn, catboost, sklearn`.

## Разведочный анализ данных

1. Удалены дубликаты данных, просмотренно количество N/A значений (в данном датасете их нет).

2. **Изучена целевая переменная `Price`.** Выведена гистрограмма и ящик с усами для всех значений, проанализирована структура даных. Выведены и проанализированны приближенныен гистограммы для большинства значений (до 3 квартиля) и самых высоких значений целевой переменной.

![Гистограмма целевой переменной](/Moscow_Housing/images/moscow_housing1.png)

![Приближенная гистограмма](/Moscow_Housing/images/moscow_housing2.png)

3. **Изучены категориальные признаки.** Выведено количество уникальных значений, обработаны дубликаты, выведены и проанализированны графики распределения.

![Диаграмма станций метро](/Moscow_Housing/images/moscow_housing3.png)

4. **Изучены вещественные признаки.** Для каждого признака выведены и проанализированны гистограммы и ящики с усами. Просмотренны части датасета с самыми большими значениями для анализа аномальных значений. Выведены данные с нереалистичными значениями.

5. **Выявлены зависимости между признаками.** Построена матрица корреляции, проанализированы линейные зависимости, удалён признак, сильно коррелирующий с другими.

![Матрица корреляций](/Moscow_Housing/images/moscow_housing4.png)

![Зависимость общей площади от жилой площади](/Moscow_Housing/images/moscow_housing5.png)


6. **Проанализировано влияние признаков на целевую переменную.** Выведены графики зависимости с вещественными признаками, сильно коррелирующими с целевой переменной, а так же несколько графиков для признаков с маленькой корреляцией. Выведены и проанализированны графики violinplot для категориальных переменных с небольшим количеством уникальных значений, а так же вивыден график зависимости цены от 20 наиболее популярных станций метро.

![Зависимость цены от ремонта](/Moscow_Housing/images/moscow_housing6.png)

## Построение модели

1. **Выполнена подготовка данных.** Выделены категориальные переменные, выборка разделена на тестовую и тренировочную, сделаны Пулы для модели Catboost.
2. **Построена модель** CatboostRegressor c функцией минимизации RMSE.
3. **Выведены метрики**. Модель оценивалась с помощью кросс-валидации с 3 фолдами.
4. **Оценена важность признаков** при обучении модели. Проанализированы пеобразования, которые в дальнейшем можно сделать с признаками для совершенствования модели.

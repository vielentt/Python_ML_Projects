# Предсказание ценообразования агрегатора такси 

В зависимости от характеристики данных было необходимо предсказать один из типов ценообразования для поездок такси, чтобы улучшить алгоритм подбора такси для клиентов сервиса. Был проведён разведочный анализ данных, построены модели многоклассовой классификации и подобраны оптимальные гиперпараметры.

## Навыки, использованные в проекте
В проекте были использованы различные библиотеки Python (Pandas, NumPy, sklearn, MatPlotLib, Seaborn). Проведён разведочный анализ EDA и построены модели классификации OvO и OvA. Ниже подробно описаны основные навыки и действия:
 - **Заполнение пропущенных значений** с помощью медианы или среднего для числовых признаков и наиболее встречающимся признаком для категориальных.
 - **Проверка корреляции признаков** с помощью визуализации матрицы корреляции и вывода топ-5 коррелдирующих признаков.
 - **Визуализация признаков по каждой целевой переменной** - "ящики с усами" и столбчатые диаграммы для анализа поведения признаков по отношению к целевой (отброс признаков без различий)
 - **Энкодинг категориальных признаков** с помощью `OneHotEncoder` и `TargetEncoder` через `ColumnTransformer` в зависимости от количества уникальных значений признаков
 - **Построение базовых моделей** `OneVsRestClassifier` на основе Логической регрессии, `OneVsOneClassifier` на осове `SGDClassifier`
 - **Скейлинг данных** с помощью `StandartScaler` и использование `PipeLine`
 - **Подбор гиперпараметров** для моделей с помощью `GridSearchCV`
 - **Вывод метрик** для сравнения качества моделей с помощью `classification_report`

# Анализ купленных вместе продуктов сети магазинов  

В данном проекте проведён ассоциативный анализ датасета транзакций с целью улучшения эффективности таргетированной рекламы магазина. Найдены пары товаров, которые покупают вместе чаще всего, а так же проведён анализ скорости работы разных алгоритмов и библиотек на датасете для оптимизации скорости анализа данных для этой сети магазинов.

## Подробнее о проекте

 - **Проведён анализ частоты встречаемости** наиболее популярных товаров. Выведены гистограммы относительной и фактической частоты
 - **Применены ассоциативные алгоритмы** 
   - Apriori с использованием трёх разных библиотек
   - FP-Growth
 - Найдены 10 наиболее встречающихся пар товаров
 - Проведено сравнение времени работы алгоритмов на датасете данного размера

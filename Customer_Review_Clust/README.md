# Кластеризация отзывов покупателей

*Для проекта использовался датасет* [Customer Opinions and Reactions](https://www.kaggle.com/datasets/ranaghulamnabi/customer-opinions-and-reactions-dataset)

Для рекламной компании магазина требовалось кластеризировать данные с отзывами клиентов. Группировка отзывов помогла адаптировать рекламу под реальные запросы и намерения пользователей.

Проведена кластеризация данных с помощью алгоритма k-means.
Использовано «правило локтя» и коэффициент силуэта для поиска
оптимального количества кластеров.

Протестированы модели с 2 и 4 кластерами

Далее была проведена кластеризация с помощью алгоритма иерархической
кластеризации и DBSCAN.

Многомерные данные были визуализированы с помощью TSNE и проведено сравнение алгоритмов кластеризации.

Файл проекта: [review_clust.ipynb](/Customer_Review_Clust/review_clust.ipynb)
